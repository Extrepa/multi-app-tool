### 1. The Color Oscillator Logic

Unlike position offsets, color oscillation requires interpolating between two or more hex values or rotating through the HSL (Hue, Saturation, Lightness) spectrum. We implement a **HSL Hue-Rotation** oscillator to create seamless, "trippy" color shifts that align with the project's aesthetic.

### 2. Implementation: The Color Vibe Utility (`src/utils/colorVibes.ts`)

This utility converts hex colors to HSL, applies the math, and returns a CSS-ready string for the SVG renderer.

```typescript
// Resource: Color-Cycle Math
export const getAnimatedColor = (baseHex: string, time: number, freq: number, amp: number) => {
  const hsl = hexToHsl(baseHex); // Utility to convert hex to HSL
  // Oscillate the Hue value
  const hueShift = Math.sin(time * freq) * amp;
  const newHue = (hsl.h + hueShift) % 360;
  
  return `hsl(${newHue}, ${hsl.s}%, ${hsl.l}%)`;
};

```
