### 1. The "Shake" Logic (Randomized Displacement)

Unlike the smooth harmonic curves of **Pulse** or **Float**, the **Shake** oscillator introduces controlled chaos into the coordinate system. It uses a pseudo-random seed to generate high-frequency displacement, perfect for "Damage" effects or environmental rumble.

* **Logic**: On every tick of the **Global  Ticker**, the object's  and  coordinates are offset by a value within a specified amplitude range.
* **Intensity Control**: The "Intensity" slider in the **Surgical Inspector** scales the maximum possible displacement, allowing for everything from a subtle nervous twitch to a violent screen-shaking impact.

---

### 2. Implementation: The Shake Function (`src/utils/vibeLibrary.ts`)

The Shake function is added to the **Vibe Engine** to handle non-sinusoidal motion.

```typescript
// Resource: Shake Logic Extension
export const ShakeOscillator = (amp: number) => {
  // Returns a random offset between -amp and +amp
  const offsetX = (Math.random() - 0.5) * 2 * amp;
  const offsetY = (Math.random() - 0.5) * 2 * amp;
  
  return { x: offsetX, y: offsetY };
};

```
