### **1. The Post-Processing Stack**

To achieve this, we wrap the **Infinite Canvas** in a WebGL-powered filter layer. Instead of modifying the SVG paths themselves, we treat the entire stage as a "texture" and run it through a fragment shader.

* **Chromatic Aberration:** Mimics the way old projector lenses struggle to focus colors at the edges, slightly separating the Red, Green, and Blue channels.
* **Luminance Bloom:** Identifies the brightest parts of your character (like Errl’s stroke) and makes them "bleed" into the surrounding darkness.
* **Film Grain / Oil Noise:** Adds a subtle, moving texture over the entire scene to replicate the organic imperfections of the original 10-year-old memory.

---

### **2. Implementation: The GLSL Shader (`src/shaders/postProcess.glsl`)**

This shader code runs on the GPU, allowing us to apply complex visual effects to dozens of instances without dropping a single frame.

```glsl
// Phase 10: Analog Post-Processing Shader
uniform sampler2D tDiffuse; // The SVG Stage Texture
varying vec2 vUv;

void main() {
    // 1. Chromatic Aberration
    float offset = 0.002; 
    float r = texture2D(tDiffuse, vUv + vec2(offset, 0.0)).r;
    float g = texture2D(tDiffuse, vUv).g;
    float b = texture2D(tDiffuse, vUv - vec2(offset, 0.0)).b;

    // 2. Add 'Oil Shimmer' Noise
    float noise = fract(sin(dot(vUv, vec2(12.9898, 78.233))) * 43758.5453);
    vec3 color = vec3(r, g, b) + (noise * 0.02);

    gl_FragColor = vec4(color, 1.0);
}

```
