# Project Structure

```
multi-tool-app/
  src/                  app source
  public/               static assets
  shared/               shared hooks and utilities
  errl-design-system/   local design system
  docs/                 documentation
  specs/                specification files
```

## Key Paths

- `src/App.tsx` - entry composition and sample project load
- `src/components/` - UI building blocks
- `src/state/` - Zustand store and types
- `src/engine/` - FX, exports, rendering logic
- `shared/utils/paper/` - vector operations
